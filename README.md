# Rock Paper Scissors Game
A simple interactive Rock Paper Scissors web game built using HTML, CSS, and JavaScript.

The game supports both button clicks and keyboard input, and the score is saved using localStorage so it remains after refreshing the page.

## Features
- Play by clicking buttons or pressing keyboard keys
- Score tracking and storing (in local storage)
- Responsive design

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Play
1. Choose Rock, Paper, or Scissors using buttons or keyboard(R,P or S).
2. The computer makes a random choice.
3. The winner is displayed and the score is updated automatically.
